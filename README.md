# COVID-19 Tracker

### Objective
* Learn the Spring framework
* Get data from APIs
* Create my own REST API
* Parse JSONs in Java

### Built with
* [Spring](https://spring.io/)

### Hosted on
* [Heroku](https://www.heroku.com/)

## App is live at
[tcovid19.herokuapp.com](http://tcovid19.herokuapp.com/)
